# LeetCode 1861 — Rotating the Box

[LeetCode (EN)](https://leetcode.com/problems/rotating-the-box/description/) ·
[力扣 (CN)](https://leetcode.cn/problems/rotating-the-box/description/)

---

## Problem / 題目

給定一個 `m × n` 的字元矩陣 `box`，代表一個盒子的**側視圖**。矩陣中每個格子為：

| 符號 | 意義                     |
| ---- | ------------------------ |
| `#`  | 石頭                     |
| `*`  | 固定障礙物（不受重力影響） |
| `.`  | 空格                     |

將盒子**順時針旋轉 90°** 後，石頭受重力影響向下掉落，直到碰到障礙物、另一顆石頭或盒子底部為止。

**回傳**旋轉並套用重力後的 `n × m` 矩陣。

---

## Solution / 解法

### 核心觀察

- **每列互相獨立**，可分別計算。
- 在原始盒子中，重力方向是**向右**（旋轉後對應**向下**）。
- 因此，可以先在原始方向模擬石頭向右掉落，再套用旋轉座標轉換——兩個步驟等效於「旋轉後再落石」。

### 演算法步驟（正序遍歷）

1. **分段**：以障礙物 `*` 為邊界，將每列劃分成多個獨立區段。
2. **計數與清除**：遍歷每個區段，以 `cnt` 統計石頭 `#` 的數量；遍歷時先將石頭視為空格寫入目標位置。
3. **填石**：當遇到障礙物或到達列末（區段結束），從尾端往前填入 `cnt` 個石頭，模擬石頭集中落至最右側。
4. **旋轉座標轉換**：同步套用公式

$$
(i,\, j) \xrightarrow{\text{順時針 90°}} (j,\; m - 1 - i)
$$

其中 $m$ 為原始矩陣的列數（row count）。

### 複雜度

| 面向     | 分析                         |
| -------- | ---------------------------- |
| 時間複雜度 | $O(m \times n)$             |
| 空間複雜度 | $O(m \times n)$（輸出矩陣）  |

## Example Walkthrough / 範例演示

以 **Test Case 2** 為例：

**輸入：**

```
# . * .     ← row 0
# # * .     ← row 1
```

### Step 1 — 正序遍歷 row 0，模擬向右掉落

掃描 `# . * .`，石頭計數 `cnt`：

| j   | 格子 | cnt | 旋轉目標位置 `ans[j][1]` | 事件               |
| --- | ---- | --- | ----------------------- | ------------------ |
| 0   | `#`  | 1   | `.`（暫時清空）          | 石頭計數 +1         |
| 1   | `.`  | 1   | `.`                     | 下一格是 `*`，區段結束 → 從 j=1 往前填 1 顆石頭：`ans[1][1]='#'` |
| 2   | `*`  | 0   | `*`                     |                    |
| 3   | `.`  | 0   | `.`                     | 列末，無石頭需填     |

### Step 2 — 正序遍歷 row 1，模擬向右掉落

掃描 `# # * .`，石頭計數 `cnt`：

| j   | 格子 | cnt | 旋轉目標位置 `ans[j][0]` | 事件               |
| --- | ---- | --- | ----------------------- | ------------------ |
| 0   | `#`  | 1   | `.`（暫時清空）          | 石頭計數 +1         |
| 1   | `#`  | 2   | `.`（暫時清空）          | 下一格是 `*`，區段結束 → 從 j=1 往前填 2 顆石頭：`ans[1][0]='#'`, `ans[0][0]='#'` |
| 2   | `*`  | 0   | `*`                     |                    |
| 3   | `.`  | 0   | `.`                     | 列末，無石頭需填     |

### Step 3 — 最終結果（旋轉後矩陣，4 × 2）

```
ans[0] = ['#', '.']
ans[1] = ['#', '#']
ans[2] = ['*', '*']
ans[3] = ['.', '.']
```

視覺化：

```
# .     ← 兩列石頭的頂部（原 col 0 → 旋轉後 row 0）
# #
* *     ← 障礙物保持不動
. .     ← 障礙物下方無石頭
```

> [!NOTE]
> 旋轉後，原始矩陣的**最右列**（j = n−1）對應旋轉後的**最底列**（row n−1），石頭向下掉落符合重力直覺。
