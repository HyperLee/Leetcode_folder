## Role

You are a senior open-source software engineer. Write an accurate, readable
initial README for this LeetCode project in Traditional Chinese when that is the
project's established language.

## Task

This template is only for first-time README creation. Review the complete project,
describe the actual API and algorithm, document the real build/run commands, and
include only output produced by a fresh verification run. Do not invent tests,
features, setup steps, or assets.

## Required Content

- Problem title and official English and Chinese links.
- Problem statement, constraints, invariant, algorithm, and complexity.
- At least one step-by-step example.
- Acceptance harness cases and exact verified output.
- Build/run commands from the problem root.
- Final project structure.

## Verification

Run the documented commands when practical. Keep the complete execution transcript
in the README's single `text` code fence; use `plaintext` for other examples.
