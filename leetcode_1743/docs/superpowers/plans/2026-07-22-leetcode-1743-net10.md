- [ ] **Step 1: Implement the minimal production API**

Use this algorithm shape:

```csharp
public int[] RestoreArray(int[][] adjacentPairs)
{
    Dictionary<int, List<int>> neighbors = [];
    foreach (int[] pair in adjacentPairs)
    {
        AddNeighbor(neighbors, pair[0], pair[1]);
        AddNeighbor(neighbors, pair[1], pair[0]);
    }

    int[] result = new int[adjacentPairs.Length + 1];
    result[0] = neighbors.First(entry => entry.Value.Count == 1).Key;
    result[1] = neighbors[result[0]][0];

    for (int index = 2; index < result.Length; index++)
    {
        List<int> candidates = neighbors[result[index - 1]];
        result[index] = candidates[0] == result[index - 2] ? candidates[1] : candidates[0];
    }

    return result;
}
```

`AddNeighbor` creates a list on first sight and appends the neighbor. Add Traditional Chinese XML summaries to both core methods and only high-signal invariant comments.
