# 3432. Count Partitions with Even Sum Difference

簡短說明
- 給定一個整數陣列 `nums`，將陣列在索引 `i` 處切割為兩個非空的連續子陣列：左子陣列為 `[0, i]`，右子陣列為 `[i+1, n-1]`。
- 請回傳所有能使左子陣列與右子陣列的元素總和差值為偶數的切分數量。

解題想法（數學）：
- 設整個陣列的總和為 `S`，左子陣列的和為 `L`。右子陣列的和為 `S-L`。
- 差值為 `L - (S-L) = 2L - S`。因為 `2L` 一定是偶數，差值的奇偶性只受 `S` 的奇偶影響。
  - 若 `S` 為奇數，差值一定為奇數，任何切分都不符合：答案為 `0`。
  - 若 `S` 為偶數，差值一定為偶數，所有 `n-1` 個有效切分都符合：答案為 `n-1`。

時間與空間複雜度
- 時間複雜度：O(n) — 需要掃描一次陣列以計算總和。
- 空間複雜度：O(1) — 只使用常數額外空間。

範例
- 輸入：`[2,1,6,4]` → `S=13` (奇) → 輸出 `0`。
- 輸入：`[1,1,1,1]` → `S=4` (偶) → 輸出 `3` (n - 1)。

注意事項
- 解法與陣列中的具體元素值無關；只需計算總和的奇偶性。
