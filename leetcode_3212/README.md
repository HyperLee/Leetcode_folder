# LeetCode 3212. Count Submatrices With Equal Frequency of X and Y

> 統計 X 和 Y 頻數相等的子矩陣數量

[English](https://leetcode.com/problems/count-submatrices-with-equal-frequency-of-x-and-y/description/?envType=daily-question&envId=2026-03-19) • [繁體中文](https://leetcode.cn/problems/count-submatrices-with-equal-frequency-of-x-and-y/description/?envType=daily-question&envId=2026-03-19)

## 題目描述

給定一個 `n × m` 的二維字元矩陣 `grid`，其中每個元素為 `'X'`、`'Y'` 或 `'.'`。

回傳滿足以下**所有**條件的子矩陣數量：

1. 子矩陣包含 `grid[0][0]`（左上角固定）
2. `'X'` 與 `'Y'` 的出現次數相等
3. 至少包含一個 `'X'`

### 範例

**範例 1:**

```
輸入: grid = [["X","Y","."],
              ["Y",".","."]]
輸出: 3
```

**範例 2:**

```
輸入: grid = [["X","X"],
              ["X","Y"]]
輸出: 0
```

**範例 3:**

```
輸入: grid = [[".","."],[".","."]]
輸出: 0
```

### 限制條件

- `1 <= grid.length, grid[i].length <= 1000`
- `grid[i][j]` 為 `'X'`、`'Y'` 或 `'.'`

---

## 解題概念與出發點

### 關鍵觀察

由於子矩陣必須包含 `grid[0][0]`，左上角已經固定在 `(0, 0)`。因此我們只需要**列舉右下角 `(i, j)`** 就能確定一個唯一的子矩陣。

> [!TIP]
> 這個「左上角固定」的限制大幅簡化了問題——不需要列舉所有可能的子矩陣，只需遍歷每個位置作為右下角即可。

### 差值計數技巧

要判斷子矩陣中 `X` 與 `Y` 數量是否相等，可以用**差值計數**：

- 遇到 `'X'` → 計數 **+1**
- 遇到 `'Y'` → 計數 **−1**
- 遇到 `'.'` → 計數 **不變**

當差值為 **0** 時，代表 `X` 和 `Y` 的數量完全相等。

### 如何高效求差值？

使用**二維前綴和**，可以在 $O(1)$ 時間內查詢任意以 `(0,0)` 為左上角、`(i,j)` 為右下角的子矩陣的差值。

---

## 解法：二維前綴和

### 資料結構設計

建立一個三維陣列 `sum[n+1, m+1, 2]`：

| 維度 | 意義 |
|------|------|
| `sum[i+1, j+1, 0]` | 以 `(i, j)` 為右下角的子矩陣中，`count(X) − count(Y)` 的差值 |
| `sum[i+1, j+1, 1]` | 以 `(i, j)` 為右下角的子矩陣中，是否包含至少一個 `'X'`（1=是, 0=否） |

> [!NOTE]
> 索引偏移 +1 是前綴和的常見技巧，用於處理邊界條件（第 0 行/列作為初始值 0）。

### 前綴和遞推公式

對於每個位置 `(i, j)`，依照二維前綴和的容斥原理：

$$
\text{sum}[i{+}1][j{+}1][0] = \text{sum}[i{+}1][j][0] + \text{sum}[i][j{+}1][0] - \text{sum}[i][j][0] + \text{val}
$$

其中 `val` 根據 `grid[i][j]` 的值：

| `grid[i][j]` | `val` | `sum[..][1]` 的更新 |
|:---:|:---:|---|
| `'X'` | +1 | 直接設為 `1` |
| `'Y'` | −1 | 繼承上方或左方是否有 `X` |
| `'.'` | 0 | 繼承上方或左方是否有 `X` |

### 判斷條件

當 `sum[i+1][j+1][0] == 0` 且 `sum[i+1][j+1][1] == 1` 時，這個子矩陣滿足所有條件，答案加一。

### 複雜度分析

- **時間複雜度**：$O(n \times m)$ — 只需遍歷矩陣一次
- **空間複雜度**：$O(n \times m)$ — 用於儲存前綴和陣列

## 範例演示流程

以 `grid = [["X","Y","."],["Y",".","."]` 為例，逐步演示：

### 初始矩陣

```
  j=0  j=1  j=2
i=0  X    Y    .
i=1  Y    .    .
```

### 逐格計算

| (i,j) | `grid[i][j]` | `sum[..][0]`（差值） | `sum[..][1]`（含 X?） | 符合條件？ |
|:---:|:---:|:---:|:---:|:---:|
| (0,0) | X | 0+0−0 **+1** = **1** | **1** | 差值≠0 |
| (0,1) | Y | 1+0−0 **−1** = **0** | 1 | **✓ 計入答案** |
| (0,2) | . | 0+0−0 +0 = **0** | 1 | **✓ 計入答案** |
| (1,0) | Y | 0+1−0 **−1** = **0** | 1 | **✓ 計入答案** |
| (1,1) | . | 0+0−0 +0 = **0** | 1 | **✓ 計入答案** |
| (1,2) | . | 0+0−0 +0 = **0** | 1 | **✓ 計入答案** |

> [!IMPORTANT]
> 上表中 `sum[..][0]` 的計算是簡化展示。完整的二維前綴和需要套用容斥公式：
> `sum[i+1][j+1] = sum[i+1][j] + sum[i][j+1] − sum[i][j] + val`

讓我們更精確地計算完整的前綴和：

```
差值前綴和 sum[..][0]:          是否含 X sum[..][1]:
     j=0  j=1  j=2                j=0  j=1  j=2
i=0 [ 1,   0,   0 ]          i=0 [ 1,   1,   1 ]
i=1 [ 0,  -1,  -1 ]          i=1 [ 1,   1,   1 ]
```

滿足 `差值 == 0` 且 `含 X == 1` 的位置：`(0,1)`, `(0,2)`, `(1,0)` → **答案 = 3** ✓
