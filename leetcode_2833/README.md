# LeetCode 2833 — Furthest Point From Origin

> 難度：Easy｜題型：字串、貪心

## 題目描述

給定一個長度為 `n`、僅由字元 `'L'`、`'R'` 和 `'_'` 組成的字串 `moves`，代表你在數線上從原點 `0` 出發的移動規則：

- 若 `moves[i] = 'L'`，**必須**向左移動一格
- 若 `moves[i] = 'R'`，**必須**向右移動一格
- 若 `moves[i] = '_'`，可**自由選擇**向左或向右移動一格

請回傳在完成所有移動後，你能距離原點的**最遠距離**。

**原題連結：**
- [LeetCode (English)](https://leetcode.com/problems/furthest-point-from-origin/description/)
- [力扣 (中文)](https://leetcode.cn/problems/furthest-point-from-origin/description/)

---

## 解題思路

### 方法：一次遍歷（貪心）

令：
- $L$ = `moves` 中 `'L'` 的數量
- $R$ = `moves` 中 `'R'` 的數量
- $B$ = `moves` 中 `'_'` 的數量

**關鍵觀察：**

在不考慮 `'_'` 的情況下，固定移動結束後位置為 $L - R$（負值表示在原點左側），與原點的距離為：

$$|L - R|$$

`'_'` 可以自由選擇方向，為了讓最終距離最大，只需讓所有 `'_'` 都朝著**與固定移動偏移相同的方向**移動。

因此，最遠距離為：

$$\text{maxDistance} = |L - R| + B$$

> [!NOTE]
> 無論 `'_'` 在字串中的位置為何，它們都可以任意選方向，因此只需統計數量即可，不需考慮順序。

**時間複雜度：** $O(n)$，單次遍歷字串  
**空間複雜度：** $O(1)$，只使用常數額外空間

## 補充說明

### 為何使用絕對值 $|L - R|$？

題目要求的是「距離原點最遠的**距離**」，而非「最終停在原點的左側還是右側」，因此結果與方向無關，只需取絕對值即可。

**$|L - R|$ 與 $|R - L|$ 完全等價：**

$$|L - R| = |{-(R - L)}| = |R - L|$$

兩者在數學上完全相同，可以互換，寫成任一形式皆正確。

### `'_'` 為何直接累加？

`'_'` 可以自由選擇方向，貪心策略是讓所有 `'_'` 全部朝著**固定移動偏移的同一方向**前進，使距離最大化：

| 情境 | 固定偏移方向 | `'_'` 最佳選擇 |
|------|------------|--------------|
| $L > R$ | 偏向左（負方向） | 全部向左 |
| $R > L$ | 偏向右（正方向） | 全部向右 |
| $L = R$ | 無偏移（位於原點） | 全部朝同一方向皆可 |

無論 `'_'` 出現在字串的哪個位置，由於可任意選方向，只需統計數量並直接累加到距離上即可，**不需考慮順序**。

## 範例演示

### 範例 1

```
Input:  moves = "L_RL__R"
```

| 字元 | L | _ | R | L | _ | _ | R |
|------|---|---|---|---|---|---|---|
| 累計 L | 1 | 1 | 1 | 2 | 2 | 2 | 2 |
| 累計 R | 0 | 0 | 1 | 1 | 1 | 1 | 2 |
| 累計 _ | 0 | 1 | 1 | 1 | 2 | 3 | 3 |

$$|L - R| + B = |2 - 2| + 3 = 0 + 3 = \mathbf{3}$$

```
Output: 3
```

---

### 範例 2

```
Input:  moves = "_R__LL_"
```

| 字元 | _ | R | _ | _ | L | L | _ |
|------|---|---|---|---|---|---|---|
| 累計 L | 0 | 0 | 0 | 0 | 1 | 2 | 2 |
| 累計 R | 0 | 1 | 1 | 1 | 1 | 1 | 1 |
| 累計 _ | 1 | 1 | 2 | 3 | 3 | 3 | 4 |

$$|L - R| + B = |2 - 1| + 4 = 1 + 4 = \mathbf{5}$$

```
Output: 5
```

因固定移動結果偏向左側（L > R），所以將 4 個 `'_'` 全部向左，可達 $-5$，距離原點 5。

---

### 範例 3

```
Input:  moves = "_______"
```

$$L = 0,\quad R = 0,\quad B = 7$$

$$|0 - 0| + 7 = \mathbf{7}$$

```
Output: 7
```

全為 `'_'`，可全部朝同一方向，距離即為字串長度。
